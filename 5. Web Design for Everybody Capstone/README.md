\# Personal Resume Website



A responsive personal portfolio website built using pure HTML, CSS, and JavaScript.  

It showcases my skills, education, projects, and experience with a clean and accessible design.

However, the content is temporary and there will be more changes.



\## Features

\- Responsive design for 4 screen sizes

\- Dark mode toggle

\- Animated skill progress bars

\- Sticky navigation with active section highlighting

\- Accessible structure (ARIA, skip link, keyboard support)



\## Tech Stack

\- HTML5

\- CSS3 (Flexbox, Grid, Media Queries)

\- JavaScript (DOM manipulation, IntersectionObserver)
